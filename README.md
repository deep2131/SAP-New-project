download-version-tagged-iflows.yml - this pipeline can download iflows which is saved as "ReadyForDeployment" but it will not show the version.
eg- in SAP CPI "Demo_iFlow" iflow saved as v.1.2.1.ReadyforDeployment it will be downloaded as "Demo_iFlow/ReadyForDeployment"
